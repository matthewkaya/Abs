## Product description (≤1200 chars)

ABS Server is an open-core, self-hostable AI orchestration platform built for developers and SMBs. It acts as the central nervous system for your AI workflows — connecting 110+ Multi-Cloud Provider (MCP) tools through a unified API, while routing inference across six providers (Anthropic, Groq, Cerebras, Cloudflare, Gemini, OpenRouter) so you optimise for cost, speed, and availability without rewriting code. RAG is built in: bring your own data, ingest it through Qdrant + BGE-M3, and ground every answer with citations and a faithfulness gate. For multi-customer setups, Cerbos PDP enforces fine-grained access control before any retrieval. Deploy via the Helm umbrella chart in 30 minutes, or run locally with Docker Compose. As a Spanish-EU company, we built data ownership in from day one — every customer can self-export via `/v1/me/data-export`, and our SOC2 audit chain is HMAC-tamper-evident. Pricing: $299 lifetime, $49/year optional maintenance, $79/month managed cloud.
